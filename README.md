# Network Security Protocols

## Common Security Protocols
- **SSL/TLS** - Secure communication over networks
- **IPSec** - Network layer security
- **SSH** - Secure remote access
- **HTTPS** - Secure web browsing

## Network Security Concepts
- **Firewalls** - Network traffic filtering
- **VPN** - Virtual Private Networks for secure remote access
- **IDS/IPS** - Intrusion Detection/Prevention Systems
- **DMZ** - Demilitarized Zone for public-facing services

## Security Layers
- **Application Layer Security** - Web application firewalls, API security
- **Transport Layer Security** - SSL/TLS protocols
- **Network Layer Security** - IPsec, VPN tunnels

## Common Network Attacks
- **DDoS Attacks** - Distributed Denial of Service
- **Man-in-the-Middle** - Intercepting communications
- **Packet Sniffing** - Capturing network traffic
- **Port Scanning** - Discovering open ports and services
